# 🕋 Qibla Direction Finder - Android App

Welcome to the **Qibla Direction Finder**, a native Android application engineered to calculate and display the accurate direction of the Kaaba (Qibla) from any location in the world.

### 🚀 Key Features
* **Sensor Fusion Technology:** Combines hardware Magnetometer and Accelerometer data to generate a stable Rotation Matrix, preventing compass distortion.
* **True North Calibration:** Automatically adjusts the magnetic declination based on the user's GPS coordinates to point towards Geographical (True) North.
* **Real-Time GPS Tracking:** Fetches live latitude and longitude to calculate the exact spherical bearing to Mecca.
* **Modern UI:** Features a sleek, circular 3D compass dial with distinct indicators.

### 🛠️ Tech Stack
* **Language:** Kotlin & XML
* **IDE:** Android Studio
* **Core APIs:** `SensorManager`, `LocationManager`, `GeomagneticField`
